# Disability IAT Studies

This repository contains two Implicit Association Test (IAT) studies examining implicit associations between disability and other social identities. The studies were developed for academic research purposes. To be added: DataPipe details.

## Studies

### Study 1: Race $\times$ Disability IAT
Examines implicit associations between race (Black people / White people) and disability (Physically Disabled People / Physically Abled People).

**Launch URL:**
[https://lowell-monis.github.io/iat/study1/](https://lowell-monis.github.io/iat/study1/)

### Study 2: Gender $\times$ Disability IAT
Examines implicit associations between gender (Women / Men) and disability (Physically Disabled People / Physically Abled People).

**Launch URL:**
[https://lowell-monis.github.io/iat/study2/](https://lowell-monis.github.io/iat/study2/)

## Study Flow

Each study follows this sequence:
1. Welcome / introduction page
2. IAT instructions (showing all four stimulus categories)
3. IAT task (7 blocks)
4. Survey questions (gender, race, partisan identity)
5. Debriefing statement
6. End page

## IAT Structure

Each IAT has four categories:
- **Category side:** Two race or gender groups, presented as photographic stimuli
- **Attribute side:** Physically Disabled People vs. Physically Abled People, presented as line-drawing icons

The IAT measures the relative speed with which participants associate each category pairing, following the standard IAT procedure (Greenwald, McGhee, & Schwartz, 1998 and Greenwald, Nosek, & Banaji, 2003).

## Stimuli

Race stimuli (Study 1): 6 Black faces (3 male, 3 female) and 6 White faces (3 male, 3 female) from Project Implicit's Race Attitudes stimulus set (Nosek et al., 2007).

Disability stimuli (both studies): 12 line drawings (6 depicting physically disabled people, 6 depicting physically abled people) from Project Implicit's Physical Disability stimulus set (Nosek et al., 2007).

Gender stimuli (Study 2): Photographs provided by the research team.

## Platform

Studies are programmed using MinnoJS (Zlotnick, Dzikiewicz, & Bar-Anan, 2015), a JavaScript framework for web-based behavioral research developed by Project Implicit. The IAT script is based on Project Implicit's MinnoJS IAT extension (Bar-Anan, 2023)
